Projects

In the past decade, I've tackled projects ranging from developing analysis routines to designing complex observations that unveil the hidden behavior of matter around black holes. But let’s stay on Earth—here are some of my most relevant projects, closely aligned with everyday industry needs.

A Python, Automated Data Reduction Pipeline for a Robotic Telescope

I was part of a small team designing, refactoring, and writing the data reduction pipeline for the ANU 2.3m Telescope. The code is reliable enough to work with a robotic telescope, triggering observations on the fly. As soon as observations are complete, the pipeline automatically detects and extracts the three brightest sources, allowing astronomers to enjoy fresh data with their morning coffee.

View Project
Highway to Spectra: A DB with Analysis Capabilities for the Largest Optical Spectral Dataset at IAC.

I designed, developed, and maintained XB-DB, a Django application for managing PostgreSQL databases. The application features a user-friendly data ingestion tool, extensive filtering options, and capabilities for querying and retrieving data. It also allows for interactive display and analysis of data in different reference frames.

GOTO-XBs: A Web App for Monitoring Black Holes and Other Cataclysmic Events.

I was the designer and lead developer of , a web application built with the Django framework. This app regularly query GOTO's databases, enabling the monitoring of galactic black holes among others.
